O que é HTML?

Estrutura Básica do HTML: HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Imagine-a como um conjunto de "blocos de construção" que você usa para organizar e dar formato ao conteúdo de um site. Cada bloco é representado por uma tag HTML. Estrutura Simples de uma Página HTML: 


<!DOCTYPE html>
<html>
   <head>
    <title>Meu Primeiro Site</title>
</head>
<body>
    <h1>Bem-vindo ao meu site!</h1>
    <p>Este é um parágrafo de exemplo.</p>
</body>
</html>

Explicação do código:

  • <!DOCTYPE html>: Informa ao navegador que o documento é um HTML5.
  • <html>: Marca o início e o fim do documento HTML.
  • <head>: Contém informações sobre a página, como o título.
  • <title>: Define o título da página que aparece na aba do navegador.
  • <body>: Contém o conteúdo visível da página.
  • <h1>: Representa o título principal da página.
  • <p>: Representa um parágrafo de texto.
  • <a>: Cria um link que leva a outra página ou site.

Para que é usado o HTML?

  • Criação de Sites: HTML é usado para construir a estrutura de sites e páginas web.
  • Estruturação de Conteúdo: Ajuda a organizar e exibir texto, imagens, e outros elementos de forma estruturada.
  • Integração com CSS e JavaScript: HTML trabalha em conjunto com CSS (para estilos) e JavaScript (para funcionalidades interativas).

Exemplos de Tags HTML Comuns:

  • <h1> a <h6>: Títulos, de <h1> (o mais importante) a <h6> (o menos importante).
  • <p>: Parágrafo de texto.
  • <a>: Link para outra página ou site.
  • <img>: Imagem.
  • <ul> e <ol>: Listas não ordenadas e ordenadas.
  • <table>: Tabelas.
